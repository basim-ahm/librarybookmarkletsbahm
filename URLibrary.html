Proxy Server Bookmarklet

The proxy bookmarklet lets you reload an open web page through the library's proxy server. If the page you are visiting is one that the library has a subscription to, and you're presently off-campus, you will get access to the resource after clicking the bookmarklet and logging in.

How Do You Add It?

To install the bookmarklet, drag this link to your bookmarks toolbar:
Read w/ UR

If you do not have a bookmarks toolbar, ctrl+shift+b on windows or cmd+shift+b on macOS will toggle the visibility of the toolbar.

How Do You Use It?

Once you've added this bookmark to your web browser, selecting it will attempt to reload your current page through the Library's Proxy Server. If the page is one that you should be able to access through the library, you should get access to it.
